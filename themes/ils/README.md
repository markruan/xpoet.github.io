# ILS

> ILS(I Love Simple)

A simple and beautiful theme for Hexo.

- [Preview](https://xpoet.cn)

## Installation

### Install

``` bash
$ git clone --depath=1 git@github.com:XPoet/hexo-theme-ils.git themes/ils
```

### Enable

Modify `theme` setting in `_config.yml` to `ils`.

### Update

``` bash
cd themes/ils
git pull
```
