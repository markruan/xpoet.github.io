## Open source projects

- **hexo-theme-keep**

  一款简约轻快的 [Hexo](https://hexo.io/) 主题，不管以后如何变化， **[Keep](https://github.com/XPoet/hexo-theme-keep)** 始终围绕「 简约、轻快、专注内容 」来维护和升级，欢迎感兴趣的同学们 PR 。关于 Keep 更多介绍以及图文教程，请移步 [Keep 官网](https://keep.xpoet.cn/) 。

  - 仓库：https://github.com/XPoet/hexo-theme-keep
  - 官网：https://keep.xpoet.cn

- **PicX**

  基于 GitHub API 开发的**图床管理神器**，图片外链使用 jsDelivr 进行 CDN 加速，**免下载、免安装**，打开 **[PicX 官网](https://picx.xpoet.cn/)** 即可使用。免费、稳定、高效，专为静态博客而生，谁用谁知道！

  - 仓库：https://github.com/XPoet/picx
  - 官网：https://picx.xpoet.cn
